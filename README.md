# The Ultimate Syteline v10 API Guide - Working Examples

This repo contains a series progressively complex Syteline API wrapper examples, written in modern .NET, meant to be used in concert with the [Ultimate Syteline v10 API Guide](https://www.linkedin.com/pulse/ultimate-syteline-v10-api-guide-part-1-overview-decision-mercer-4ba3c/) series of articles, published on LinkedIn.

To try it out, clone the repo, enter you credentials for whichever method you want to use (direct or ION), and run the program.
